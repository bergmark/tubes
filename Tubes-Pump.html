type Pump a b = CofreeT (PumpF a b)

A Pump is the dual to a Tube: where a Tube is a computation manipulating a stream of values, a Pump can be situated on either end of a tube to both insert values when requested and handle any yielded results.

class (Functor f, Functor g) => Pairing f g | f -> g, g -> f where

Lovingly stolen from Dan Piponi and David Laing. This defines a poor man's adjunction: it allows adjoint functors to essentially annihilate one another and produce a final value.

If something equivalent comes along, I'll switch to it.
